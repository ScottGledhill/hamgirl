# Hamgirl

Hamgirl was created in honour of a girl I know who loves ham.

To play you must:

Brew install sdl2
```
bundle install
```
To launch the game
```
ruby game.rb
```

# Controls
Use the arrow keys to move up/down/left/right to chase the ham
